How it works (4 specialized AI agents working together):

 Librarian Agent → Searches Semantic Scholar + arXiv for relevant papers
 Reader Agent → Analyzes abstracts, extracts methodologies & findings
Strategist Agent → Compares papers & identifies research gaps
 Writer Agent → Generates a complete thesis-ready report

 Tech Stack:
• CrewAI — Multi-agent orchestration
• Streamlit — Interactive dashboard
• Semantic Scholar API + arXiv API — Paper retrieval
• Groq Llama 3 — Fast, free LLM (14k requests/day!)

 What it delivers:
 Systematic literature review with abstract
 Comparative analysis of methodologies
 Research gaps with evidence
 Future research directions with timelines
 Downloadable Markdown report
